2. Formula Evaluation Architecture

Engine tính toán động độc lập. Xử lý Abstract Syntax Tree (AST) và Validation nghiêm ngặt.

  • Context Injector: Nạp các biến số như weight, height, age, bmr từ PatientProfile.
  • Validation Rules:
    • Nếu biến bị null/missing → Raise MissingContextVariableError. Engine sẽ fallback về cấu hình mặc định an toàn hoặc chặn đánh giá.
    • Nếu kết quả negative (âm) khi tính ngưỡng dinh dưỡng → Ép về 0.0 và ghi log Warning (tránh tính ra Calorie âm).
# Example AST for: 0.6 * weight
FormulaAST(
    operator="MULTIPLY",
    left=Literal(0.6),
    right=Variable("weight", fallback=None, required=True)
)

3. Category Hierarchy & Ontology Rules

Quy tắc phân cấp thực thể và giải quyết sự nhập nhằng (Ambiguity) trong Knowledge Graph.

  • Multiple Parents: Một Ingredient có thể thuộc nhiều Category.
  • Canonical Parent (BẮT BUỘC): Phải có đúng 1 Category đóng vai trò là cha chính (Primary Path). Ví dụ: Tomato → Canonical: Fruit | Secondary: Vegetable. Điều này giúp tính điểm trọng số và truy vết đồ thị mượt mà hơn.
  • Cycle Prevention: Topo-sort DAG validation trước khi lưu.

4. User Preference & Meal Context Model

Mô hình hóa sở thích cá nhân và bối cảnh bữa ăn (Meal Context) để hỗ trợ Recommendation.

class UserPreferenceProfile:
    disliked_ingredients: List[str]  # e.g., ["Coriander", "Onion"]
    preferred_cuisines: List[str]    # e.g., ["Vietnamese", "Japanese"]
    dietary_style: str               # e.g., "Vegetarian", "Vegan", "Omnivore"
    religious_restrictions: str      # e.g., "Halal", "Kosher"

class MealContext(Enum):
    BREAKFAST = "BREAKFAST" # Thường ưu tiên Carbs nhanh, nhẹ bụng
    LUNCH = "LUNCH"         # Đủ Macros
    DINNER = "DINNER"       # Ưu tiên Protein, ít Carbs
    SNACK = "SNACK"         # Calorie thấp

5. Recommendation Scoring Specification

Công thức xếp hạng (Ranking Formula) đa chiều, loại bỏ hoàn toàn tính chất "Black Box" của AI.

$$Total\_Score = (0.4 \times S_{safety}) + (0.25 \times S_{budget}) + (0.2 \times S_{goal}) + (0.1 \times S_{pref}) + (0.05 \times S_{variety})$$
